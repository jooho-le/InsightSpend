# InsightSpend

## 프로젝트 소개
InsightSpend는 근로자의 개인 맞춤형 스트레스 관리를 목표로 하는 서비스입니다.
사용자가 기록한 스트레스/감정 데이터와 소비(가계부) 데이터를 함께 저장·분석하여, “스트레스가 소비 습관에 어떤 영향을 주는지”를 직관적으로 이해하고 건강한 스트레스 해소 방식과 올바른 소비 습관을 형성하도록 돕습니다.

본 프로젝트는 Web App(React) 과 Mobile App(React Native/Expo) 을 직접 구현하고, 두 앱을 동일한 Firebase 프로젝트(Authentication + Firestore) 에 연동하여 데이터를 공유하는 것을 목표로 합니다.
Web App은 Firebase Hosting이 아닌 JCloud 환경에 배포합니다.

문제 해결 아이디어

스트레스가 높아질수록 충동 소비로 이어지는 경우가 많습니다.
InsightSpend는 단순히 감정과 지출을 따로 기록하는 앱이 아니라, 다음과 같은 연결을 중심으로 설계했습니다.

기분이 안 좋은 날/스트레스가 높은 날에 지출이 증가하는지

특정 감정 상태에서 소비 카테고리(식비/쇼핑/교통 등)가 바뀌는지

스트레스가 높은 날 vs 낮은 날의 소비 규모 차이

이를 기반으로 사용자가 스스로 패턴을 인지하고, “지출로 스트레스를 푸는 방식” 외에도 호흡·명상·가벼운 운동·휴식 루틴·커뮤니케이션 팁 등 대체 방법을 선택할 수 있도록 유도합니다.

## 주요 기능
1) 로그인/인증

Firebase Authentication 기반 로그인(소셜 로그인 1종 이상)

사용자별 데이터 접근(본인 데이터만 읽기/쓰기)

2) 스트레스/감정 기록

날짜별 스트레스 점수 및 감정(기분) 기록

기록 조회/수정/삭제

3) 가계부(재정) 기록

수입/지출 기록, 카테고리 분류

기록 조회/수정/삭제

4) 스트레스 × 소비 패턴 분석(인사이트)

“스트레스 높은 날 vs 낮은 날” 지출 규모 비교

감정/기분별 상위 소비 카테고리 요약

기간별 요약 인사이트 생성 및 저장(예: 일간/주간/기간 요약)

5) 근로자 공감형 AI 챗봇

사용자가 고민/상황을 입력하면 공감형 대화로 정서적 지지 제공

(선택) 직종(서비스직/생산직 등) 선택 시 업종 특성을 고려한 응답

대화 내용을 바탕으로 맞춤형 스트레스 해소 활동 제안

(선택) 대화 로그 저장을 통한 개인화

6) AI 기반 코멘트(선택 기능)

최근 기록을 바탕으로 “스트레스 상태 + 소비 패턴”을 한 문장 요약

과소비 패턴이 감지되면 대체 루틴(휴식/운동/명상 등) 추천

간단한 설치·실행 가이드입니다.

## 설치
1. 리포지터리 루트에서 의존성을 설치합니다.

   ```bash
   npm install
   ```

2. 워크스페이스(`apps/mobile`/`apps/web`)마다 따로 `npm install`이 수행되므로, 루트 install만으로 충분합니다.

## 실행
### 모바일 (Expo)
1. `.env`(apps/mobile/.env)에서 Firebase/Google/AI 환경변수를 채운 후 Expo 서버를 시작합니다.

   ```bash
   npm run dev:mobile
   ```

2. QR 코드 또는 시뮬레이터로 Expo 앱을 실행하고, Google 로그인을 테스트합니다.

### 웹 (Vite)
1. `.env`(apps/web/.env)에서 `VITE_OPENAI_*` 등을 채운 후 웹 서버를 실행합니다.

   ```bash
   npm run dev:web
   ```

2. `http://localhost:5173`(또는 Vite가 안내하는 URL)에서 대시보드를 확인하세요.

## 참고
- Firebase 설정(`EXPO_PUBLIC_*`, `VITE_*`)이 정확해야 Google 로그인 / AI 기능이 동작합니다.  
- 배포 시 환경변수를 호스팅 서비스(예: JCloud, Vercel)에 다시 등록하는 것을 잊지 마세요.
